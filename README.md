## 👤 Users APIs
| API Name       | Method | Endpoint                | Example URL                                        | Description               | Auth  |
| -------------- | ------ | ----------------------- | -------------------------------------------------- | ------------------------- | ----  |
| Get All Users  | GET    | `/users/`               | `https://apis-z5tu.onrender.com/users/`            | Retrieve all users        | ❌ No |
| Get User by ID | GET    | `/users/id/:id`         | `https://apis-z5tu.onrender.com/users/id/1`        | Get a specific user by ID | ❌ No | 
| Get by Salary  | GET    | `/users/salary/:salary` | `https://apis-z5tu.onrender.com/users/salary/1000` | Filter users by salary    | ❌ No |
